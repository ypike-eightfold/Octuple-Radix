
## Usage in Code

Access design tokens using CSS variables:

```css
.my-component {
  color: var(--indigo-9);
  background: var(--gray-1);
  border-radius: var(--radius-3);
  padding: var(--space-4);
  box-shadow: var(--shadow-2);
}
```

Or use Radix Themes props:

```tsx
<Box p="4" style={{ background: 'var(--indigo-3)', borderRadius: 'var(--radius-3)' }}>
  <Text size="3" weight="medium">Content</Text>
</Box>
```
