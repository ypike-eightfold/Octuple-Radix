## What is Octuple Radix?

**Octuple Radix** is the third version of the Octuple Design System, built on top of **Radix Themes** - a comprehensive, accessible, and customizable component library. It combines the power of Radix's production-ready components with Eightfold AI's design language.

## Quick Start

Get started with Octuple Radix in three simple steps:

```bash
# 1. Install the package
npm install @eightfold.ai/octuple-radix

# 2. Import styles in your app
import '@eightfold.ai/octuple-radix/styles.css'

# 3. Wrap your app with ThemeProvider
import { ThemeProvider } from '@eightfold.ai/octuple-radix'
```

```tsx
function App() {
  return (
    <ThemeProvider>
      <YourApp />
    </ThemeProvider>
  )
}
```

## Component Categories

Octuple Radix includes 60 components organized into these categories:

- **Form Controls** - Buttons, inputs, selects, checkboxes, radio buttons, switches
- **Data Display** - Tables, cards, badges, avatars, data lists
- **Navigation** - Breadcrumbs, pagination, tabs, menus, navigation bars
- **Overlays** - Dialogs, popovers, tooltips, modals, toasts
- **Layout** - Flex, grid, containers, sections
- **Typography** - Headings, text, links, code blocks
- **Feedback** - Progress bars, spinners, empty states, alerts
